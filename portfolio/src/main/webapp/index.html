Instruction-level and thread-level parallelism. Data-level parallelism and graphics processing units. Multiprocessors. Cache coherence. Interconnection networks. Multi-core processors and systems-on-chip. Platform architectures for embedded, mobile, and cloud computing.

Advanced Logic Design

Attending Class for Fall 2020. Another one of the big bulls in the CE course list. Some of the topics covered include model, simulating, synthesizing and optimizing large and complex subsystems It also seamlessly covers some of the practical industrial aspects of modern design, including use of hardware description languages (e.g. VHDL) for structured modelling and simulation. Other topics include: controller synthesis and optimization, iterative circuits, high-speed combinational arithmetic circuits,fault-tolerance and soft error mitigation, power optimization strategies, asynchronous design, FPGA struc-tures, and floating-point arithmetic. Will also gain hands-on experience in designing and simulating a number of real digital systems using several CAD tools.

Circuit Analysis

Network theorems. Transient response of first and second order circuits. Sinusoidal steady state analysis. Frequency response of linear circuits. Poles and zeros. Bode plots. Two-port networks.

Operating Systems

Class attended Spring 2020. Easily one of the hardest classes I have ever taken. The class focused on hacking the Debian Linux OS. In order to do that we had to learn in depth how diffferent parts of the OS worked together. We learned about schedulers, task-structs, synchronis and asynchronis programming, and implementing system calls just to name a few. The class had a "self exploration mind set", where we learned about concepts and theories and class and then had to dive into linux documentation on our own to find out "how to build and implement another scheduler"? Or how to add another file-system into our OS?

projects worth bragging about (C): Building a multi-server that can handle HTTP protocol. Building an OS scheduler called freezer. Implementing an OS wide system call called fridge which was basically a gloabl hash table Implementing a custom file-system
